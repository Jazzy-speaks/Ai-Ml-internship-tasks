# House Price Prediction

## Objective
Predict house prices using features like area, bedrooms, and location.

## Dataset
House Price Prediction Dataset (Kaggle)

## Models Used
- Linear Regression
- Gradient Boosting Regressor

## Results
- Gradient Boosting performed better than Linear Regression
- Lower MAE and RMSE achieved with Gradient Boosting

## Key Features
- Area
- Location
- Condition
- Number of bedrooms and bathrooms

## Conclusion
The model successfully predicts house prices and shows that property size and location are key factors.
